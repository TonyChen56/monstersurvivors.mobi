About Monster Survivors

Monster Survivors is an action-packed roguelike survival game where you must fight against endless waves of monsters. Each run is unique, with random enemy spawns, power-ups, and challenges to overcome.

Inspired by games like Vampire Survivors, this game combines fast-paced action with strategic character building. How long can you survive as the difficulty continuously increases?

Choose your character, collect powerful weapons and upgrades, and see if you can survive the monster apocalypse. With each run, you'll learn new strategies and discover powerful combinations to help you last longer against the relentless hordes.

Game Features

Endless Survival

Face increasingly difficult waves of monsters in an endless survival challenge that tests your skills and strategy.

Character Progression

Level up during each run, unlocking new abilities and becoming more powerful as you fight.

Unique Weapons

Discover and upgrade a variety of weapons, each with different attack patterns and special effects.

Random Encounters

Every run is different with procedurally generated enemy waves and power-up opportunities.

Game Controls

Master these controls to survive longer against the monster hordes:

W Move Up
A Move Left
S Move Down
D Move Right
Space Special Ability
ESC Pause Game

You can also use arrow keys for movement. On mobile devices, touch controls will automatically appear.
